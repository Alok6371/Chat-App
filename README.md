# Chat-App

## Uses and Functions

### express
- **Function**: Fast, unopinionated, minimalist web framework for Node.js.
- **Uses**: To create the server and handle routing.

### cors
- **Function**: Middleware to enable Cross-Origin Resource Sharing (CORS).
- **Uses**: To allow or restrict requested resources on a web server depending on where the HTTP request was initiated.

### mongoose
- **Function**: Elegant MongoDB object modeling for Node.js.
- **Uses**: To interact with MongoDB, define schemas, and perform CRUD operations.

### jsonwebtoken
- **Function**: JSON Web Token implementation for authentication.
- **Uses**: To create and verify tokens for user authentication.

### cookie-parser
- **Function**: Middleware to parse cookies.
- **Uses**: To parse and manage cookies in the request object.

### dotenv
- **Function**: Module to load environment variables from a `.env` file.
- **Uses**: To manage configuration settings and secrets.

### bcryptjs
- **Function**: Library to hash passwords.
- **Uses**: To securely hash and compare passwords for user authentication.
